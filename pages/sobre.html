Sobre o Nosso Projeto

A Ideia

Este site foi desenvolvido como parte do Challenge de Front-End Design Engineering. O objetivo foi criar uma página institucional estática para a ONG "Turma do Bem", a maior rede de voluntariado especializado do mundo. Escolhemos esta organização por seu imenso impacto social, devolvendo o sorriso e a dignidade a milhares de pessoas.

Tecnologias Utilizadas

Para a entrega do Sprint 01, focamos em construir uma base sólida e visualmente coesa, utilizando exclusivamente as seguintes tecnologias web fundamentais:

  • Utilizado para a estruturação semântica de todo o conteúdo do site, garantindo acessibilidade e organização.
  • Empregado para a estilização completa, incluindo a paleta de cores, tipografia, layout em Flexbox e o design dos componentes, seguindo o breakpoint de 1300px para desktop.

Roadmap

Este é apenas o primeiro passo. Nossos planos para as próximas Sprints incluem:

  • Sprint 02: Implementação da responsividade para garantir uma experiência de usuário otimizada em tablets e dispositivos móveis.
  • Sprint 03: Adição de interatividade com JavaScript para validação de formulários, animações e uma navegação mais dinâmica.
  • Sprint 04: Exploração de integrações com APIs ou um back-end simples para funcionalidades como cadastro de voluntários ou simulação de doações.

Sobre o Nosso Projeto

A Ideia do Projeto

Este site foi criado para o Challenge de Front-End Design da faculdade. Nossa equipe escolheu desenvolver uma página para a ONG "Turma do Bem" devido ao seu incrível impacto social, que leva saúde bucal e autoestima para milhares de pessoas.

Tecnologias Utilizadas (Sprint 01)

Nesta primeira fase, construímos um site estático focado na experiência para desktop, utilizando as tecnologias essenciais da web:

  • HTML5: Para estruturar todo o conteúdo das páginas.
  • CSS3: Para criar o design, layout, cores e fontes, garantindo uma aparência profissional e consistente.

Próximos Passos (Roadmap)

Para as próximas Sprints, nosso plano é evoluir o projeto com as seguintes melhorias:

  • Tornar o site responsivo: Adaptar o layout para que funcione perfeitamente em celulares e tablets.
  • Adicionar interatividade: Usar JavaScript para criar animações e validar o formulário de contato.
  • Criar funcionalidades dinâmicas: Integrar o site com um sistema para gerenciar voluntários e doações de forma real.
